## 旅途 {#旅途}

虽然本文是游记，但故事（或者说事故）最多的，是旅途。

机票是11月12日定的，中国外交部11月14日提醒公民今日避免赴日，不过查了一下，高市的言论在11月7日就说了，
这说明我毫无政治敏感性。但无论如何，除了开战这种级别的事情，没有什么能阻止我出行，因为买的廉航机票不能退。

我计划在11月28日凌晨两点出发。因为是第二次去日本，所以我做好了准备：买了一件[廉航神器](https://e.tb.cn/h.SrOBdvJCjXXpWwC?tk=6hOPfiv577m%20)；带了一个零钱袋；
之前团建别的团队送了一个可以揣口袋里面的小保温杯；在 tabelog 上定了一顿晚饭；
照着京都的旅游指南定了旅游计划；买的相机也能顺利使用。「万事俱备」的感觉让我心情越发愉悦，
可以说整个人都沉浸在期待中。

事情在11月27日19:30左右出了变化，我在下班回家的列车上收到 Agoda 的邮件，Peach 航空取消了航班[^fn:1]，
我当时就慌了（其实不用慌，因为慌也没有用），冲回家确定了 Peach 不能给我换一班机，以及试图联系 Agoda
客服无果之后，我忍痛买了东航当晚23:00出发的机票，贵了两百多不说，我还得在浦东转机。这时候已经8点多了，
幸好昨晚已经打包好行李，航班23:00起飞我就不能坐 MRT 只能打车了。

到了樟宜机场，我还无法自动值机，只能走柜台，心头又是一紧。直到拿到登机牌，我才放松下来。在机场的 NamNam
吃了一顿 Banh Mi Op La Beef Steak &amp; Eggs. 极度紧张之后放松让食物变得更美味——不过确实这道菜还行，
牛排煎得可以，不是能上路的状态，半凝固的煎蛋用来蘸烤过的法棍也是不错的体验，越南人怎么评价我不清楚，
在新加坡这种平价美食荒漠中的厕所绝对是满分佳作。

打车去机场的路上我抽空跟父母说了我去京都旅行，我妈没说什么，我爸过来跟我说为什么这个时候去日本，
不看看现在中日关系紧张么？我说那我见了高市早苗跟伊说一句，你要见了王毅也帮我提一嘴。我爸不答我这下茬，
叮嘱我注意安全就算了[^fn:2]。

{{< figure src="/ox-hugo/pudong-sunrise.jpeg" caption="<span class=\"figure-number\">Figure 1: </span>浦东机场日出" >}}

2021年我上海转机去新加坡，我忘记当时是从虹桥还是浦东起飞的了。从浦东出发之后，还算顺利。降落关西之后，
买了 JR Haruka 票——这个时候我脑抽了买了站票，站了一个多小时，后来回来的时候我就买座了。
坐到京都站换乌丸线。酒店在五条站(Gojo)，但是学过一点点日语的我把九条(Kujo)当五条了，坐反了一次，
最后到酒店的时候都四点多，睡了一觉，本来计划28号上午逛东西本愿寺的计划全取消了。


## 11月28日 {#11月28日}

如前所述，因为太阳黑子，我28日的行程只剩下一个：吃晚饭。餐厅叫ジビエの隠家(Gibier no Nabariya)，
27号在 tabelog 上预订的（特别搞笑的是他们家只有晚饭，但是放了中午的预订 slot ，害我订了两次，
不过也好，如果订了午饭，那我的预订费用就亏了）。

大约6点多，我从酒店出门，天已经黑了，走在居民区的小路上，安静昏暗。之后我还会多次走在这样的路上。
这家店是个和式西餐厅，前面被一家居酒屋挡住了，不仔细找还真找不到。在 Google translate 的帮助下，
我顺利跟店家表达了「除了预订的套餐之外，还想吃点熊肉。」店家也很愉快地告诉我：「熊肉没货了，整点别的吧。」
最后我点了一道狸猫[^fn:3]。

鹿肉新加坡也能吃到，但羌肉和狸猫肉就很难得了。这道狸猫肉味道也不错，虽然筋膜没有去掉，
但因为我牙齿健康，也没所谓了，香味很强烈。吃完之后饱了。

然后我换了一条路步行回酒店，带上我的廉航神器马甲，我穿了四件衣服，风吹来还是有点冷。
路上发现有家文具店（30日去了），8点关门的，后面白天再去看看。

京都公路交通灯用了一种很像鸟鸣的声音作为切换提示，害我以为是树上有鸟。


## 11月29日 {#11月29日}

决定去岚山竹林小径，然后从岚山去金阁寺，最后去下鸭神社。

在京都站换乘，我依然头大，幸好有个高瘦的阿伯指路。这阿伯也是去岚山，上车钱我买了一个便当，
我问阿伯能不能吃便当，伊说不能，我也就算了（后来我坐 Haruka 去机场，旁边的日本人吃得不亦乐乎）。
阿伯还指点我装镜头（第二次用，还没有熟练）,感谢阿伯。

岚山这竹林，也就那样了，但岚峡挺漂亮的。

从岚山公园下来，观光火车时间最近一趟已经是下午一点半了，就不买了（假如还有下次，一定要预订）。
在小火车站前买了串三彩团子[^fn:4]，不好吃，非常后悔，而且这花了我200日元，导致后面买天龙寺入场券的时候，
我要破开我的10000日元大钞。

然后我在小仓池看到了绿头鸭，今年开始观鸟之后，去哪都会留意一下当地的鸟，这次没有带望远镜，甚是可惜。

{{< figure src="/ox-hugo/mullard.jpeg" caption="<span class=\"figure-number\">Figure 10: </span>绿头鸭" >}}

在小仓池转悠一圈之后去了天龙寺，无甚足观，庭院中植物有点意思。我记了一个列表。豆柿正在结果，
我还真的挺想尝尝的，出门的时候，有个牌子写着豆柿是日本特有的。

{{< figure src="/ox-hugo/Enkianthus_perulatus.jpeg" caption="<span class=\"figure-number\">Figure 11: </span>日本吊钟花这个日本名字灯台踟躅有点意思" >}}

{{< figure src="/ox-hugo/Diospyros_lotus.jpeg" caption="<span class=\"figure-number\">Figure 12: </span>豆柿能吃吗？" >}}

从天龙寺出来我坐岚电线(Randan)去金阁寺，岚电线是京都当地的铁路线，没记错的话应该只有三节车厢，
慢悠悠地穿过京都，座椅已经包上棉垫，车厢开了暖气，挺暖和的。

金阁寺门口就是著名的大文字山，从《有顶天家族》知道五山送火，如意岳大文字山就是天狗药师坊的根据地。

{{< figure src="/ox-hugo/nyoigatake.jpg" caption="<span class=\"figure-number\">Figure 13: </span>光秃秃的如意岳大文字山，可惜上面没有天狗。" >}}

我没有读过三岛由纪夫（坦白说也提不起兴趣，川端康成的《古都》我努力了，也没读下来），
对金阁寺的兴趣主要是《妄想学生会》里面，会长的修学旅游去了金阁寺，我可以拍张圣地巡礼。
金阁寺这金光四射的，俗气，还不如静安寺，虽然静安寺也很俗，但好歹有点气势，
金阁寺这孤零零立在日式庭院里面，很突兀。

{{< figure src="/ox-hugo/wangxiangxueshenghui.jpeg" caption="<span class=\"figure-number\">Figure 14: </span>我觉得角度差不多了，不过不好意思指挥拍照的欧巴桑蹲下来用仰角。" >}}

巡礼完我沿路往上走，在山顶的礼品店买了一个金阁寺的冰箱贴。这次来，毫无搞封建迷信的兴致，
参拜和抽签都省了，旅游文创有些还是值得买的。不过出门的时候两欧巴桑在卖加了金箔的抹茶，
我尝了一口，甜得发腻。

{{< figure src="/ox-hugo/seven-gods.jpg" caption="<span class=\"figure-number\">Figure 15: </span>但是我把所有硬币（三枚）留在这里了，男人，呵！" >}}

走出金阁寺已经是午饭时候，搜了一下 tabelog ，附近没啥想吃的，干脆直接坐公交去下鸭神社。
公交站前还有耶和华见证人教会的人在传教，见我是中国人，还有找了个能说中文的人过来，
我应付两句就去车站了。

出了车站，在街上逡巡一会，找了家很「小红书风」的荞麦面店，进店左手边有张桌子，摆了一些美食书籍，
可能作者还宣传过这家店。店里面是一张大方桌，客人围着方桌自己找位置，我点了一份1000日元的荞麦面天妇罗套餐（菜单上的套餐种类很少）。
吃完之后，觉得不功不过。吃完还顺路去了一家精品咖啡店（京都下鸭焙煎所，
shimogamo baisenjo），点了他们家的中烘的豆子，喝不出特别的意思。上次去东京没尝日本的手冲咖啡，因为当时不想用现金，
这次去京都，我做好准备了，带了个零钱包——而且感觉京都支持 contract less 支付的店更多。

{{< figure src="/ox-hugo/Shimogamo-Shrine1.jpeg" caption="<span class=\"figure-number\">Figure 17: </span>《有顶天家族》巡礼其二" >}}

下鸭神社里面在做一个国宝巡展，我问一个巫女（？），多少钱？伊回答 One Thousand 的时候我愣住了，
我期待伊说「一千円」的呀，不管如何，我嫌贵，没有进去，于是往纠之森走去。

纠之森里面有个小型演唱会，有个歌手在签售伊的 CD ,是个不太知名的歌手，我当时扫了伊的推特，
但是没有关注，现在找不回来了。

{{< figure src="/ox-hugo/Tadasu-no-Mori.jpg" caption="<span class=\"figure-number\">Figure 19: </span>纠之森内的演唱会" >}}

然后我去了鸭川公园，Kamogawa Delta 那里看鸭川和 Takano River 交汇处，远处有几只鹰在盘旋。
可惜没有带望远镜（此后我会多次惋惜没带望远镜）。

在 Kamogawa Delta 那里看小孩玩水，先从 Stepping Stones 跳到鸭川左岸，然后再跳回右岸，
这个 Stepping Stones 出现在《有顶天家族》第二季《二代目归来》的海报上。

{{< figure src="/ox-hugo/youdingtian.png" caption="<span class=\"figure-number\">Figure 20: </span>老实说，我不太喜欢《有顶天家族》第二季" >}}

跳到右岸之后，我想去附近看看织田信长的墓，虽然我觉得肯定会失望——然后我果然失望了，
此事后文再叙。穿进居民区的小路，看着这些早就从动漫日剧熟悉的一户建民居，
我有种淡淡的哀愁——非常适合日本幽玄物哀的审美。

当然这哀愁不是很浓，因为我当时面临一个问题：尿急。我当时想了两种解决方案：

1.  信长墓肯定是个景点，景点就有厕所。
2.  「只要不被抓到，就不算犯罪」。

路过一个小停车场的时候，我非常走运，看到了一群黑鸢。这群黑鸢跟一群大嘴乌鸦混在一起，
乌鸦真不愧是日本国鸟啊，倍儿有面，前面我追着一只盘旋的鸟，追到马路边才发现是乌鸦，
看到这混居的鸟群我就明白了，感情天上盘旋的时候就混着呢。

{{< figure src="/ox-hugo/black-kite.jpg" caption="<span class=\"figure-number\">Figure 21: </span>跟乌鸦混在一起的黑鸢" >}}

最后拐到信长墓那条小路，旁边有个小游乐场——《地狱老师》里面经常闹鬼的地方，果然有厕所，
我不用做降低道德底线的事情[^fn:5]。

为什么信长墓让我失望呢，不是因为这景点没啥看的，也不是因为天黑了景点已经关门了，
而是那破景点被一个阿弥陀寺包了，居然要500日元!我从停车场进去拍张远处山景红叶得了。

{{< figure src="/ox-hugo/from-emituosi.jpg" caption="<span class=\"figure-number\">Figure 22: </span>不过其实当时我闯进去，估计也没人拦我" >}}

这时天已经开始黑了，我往地铁站走去，觉得非常寂寞，我希望有人能陪伴我，但不是任何一个现实中认识的人，
在我写下这段话的时候[^fn:6]我在看石黑正数漫画改编的电影《睡觉的笨蛋》，我希望有个像石黑正数漫画里面的角色在身边。
坦白说，这次旅途中，我屡次想象，如果我能在京都长大就好了，有一群从小学开始就在同样的环境一起长大的伙伴，或许就不会觉得寂寥。
当然我很清楚这种梦幻是永远达不到的彼岸，人始终是嵌入在社会结构里面的，这种梦幻是因为只看到美好的一面——人生的无奈就是利弊往往都是一体两面的。

回到酒店，我故技重施，问了一下晚上有什么活动[^fn:7]，前台告诉我晚上东寺开放。我回到房间躺了一会，觉得很累，但犹豫了一下，起来去东寺。
晚上去参拜的人很多，我都要打退堂鼓了，但还是掏了1000日元进去。但不得不说，东寺值得这个钱。东寺的红叶，在夜灯的映照下，灿烂辉煌，
五重塔巍峨耸立。金殿庄严，我排队进去看了里面的佛像，里面不允许拍照（但允许装大灯照得亮如白昼）。无论如何，看了点真古董，我觉得这1000日元很值。

{{< figure src="/ox-hugo/east-gate.jpeg" caption="<span class=\"figure-number\">Figure 24: </span>东寺东大门，又名「不开门」，背后的故事请自行搜索" >}}

回来泡完澡，发现我这一天走了30000多步，决定明天休息一天，对，在休假中休息。


## 11月30日 {#11月30日}

这天起来，首先是看一下最近的电影院，然后看了一下上映的电影，有《链锯人：蕾塞篇》，好，网上买票。

然后出门，去鸭川边上溜达一趟，去的路上进佛光寺转了一圈，有个小姑娘在银杏树下拍，我帮伊抓两把落叶去洒，
回来途中去了一家叫 TAG 的文具店，买了一套三款鸟类主题的书签[^fn:8]，还别的一些文件，凑够了免税。

{{< figure src="/ox-hugo/bukkoji.jpeg" caption="<span class=\"figure-number\">Figure 27: </span>佛光寺内的银杏" >}}

{{< figure src="/ox-hugo/kamo-river.jpeg" caption="<span class=\"figure-number\">Figure 28: </span>秋日水孱缓，友人行迹未曾休，最鸭川" >}}

回到酒店在一楼酒吧点了杯酒，跟酒保聊了一会，说了我学日语的事情，让伊推荐午饭的地点，伊推荐拉面横纲，
我要还是东京那时候的性格，不到山穷水尽怎么会吃这种连锁品牌?不过我这次也算了，喝完酒过去吃了一顿，
在路口一望，京都三面环山，挺有风致的。

{{< figure src="/ox-hugo/chainsawman.jpeg" caption="<span class=\"figure-number\">Figure 29: </span>蕾塞！蕾塞啊！" >}}

下午去看完电影，刚好之前在豆瓣看到的 [Bar Nostalgia](https://maps.app.goo.gl/ELufbjbRTFkLp9Z47) —— 森见登美彦爱去的酒吧，去看了伊的前面，上面有留言簿，
大量森见粉丝的留言，我喝了一整套《有顶天家族》的酒单：只推荐弁天和矢三郎。

{{< figure src="/ox-hugo/morimi.jpeg" caption="<span class=\"figure-number\">Figure 30: </span>森见的签名" >}}

喝酒的时候我英语夹带日语跟酒保搭话，有意思的是那个主要做招待工作的年轻酒保没读过森见的书，我为什么酒单上只有矢二郎和矢三郎，

## 12月1日 {#12月1日}

去宇治！看了一个小红书攻略，感觉可以玩个大半天，今天的重头戏其实是晚餐，我预约了一家可以吃熊肉的居酒屋。

下车先去了小红书推荐抹茶连锁，里面的服务员规矩大得很，叽里咕噜说了一通，大意是得点单才能坐啥的，而且还得先拿票，
如果「来都来了」，同时没吃早饭，我就走了。菜单上有我喜欢的要乐奈排名仅次于 live 、吉他和奶奶的最爱——抹茶芭菲，
但我的肠胃告诉我并不想空腹服用一大杯冰淇淋，最后我选了一份抹茶丸子和一杯抹茶[^fn:9]，垫吧垫吧免得低血糖得了。

{{< figure src="/ox-hugo/mochabafei.jpeg" caption="<span class=\"figure-number\">Figure 35: </span>乐奈，爹想不明白你怎么框框暴吃这玩意儿的" >}}

吃完就去平等院。平等院里面有一块源赖政的碑，用汉字写的，大概内容应该是把源赖政比作荀彧，把平家比作曹操（碑中称曹瞒）？
平安时期历史不熟，兴趣不大。平等院凤凰堂值得一去，理由跟东寺一样：阿弥陀如来像是真古董。
话说这种末世思想是不是日本人的「自古以来」？跟物哀美学挺搭的。附近还有源氏物语博物馆，但我没读过这书，不去了。
更有意思的是，凤凰堂里面的云上供养观音有52个，所以他们做了一副扑克，如此巧合！我心服口服，不得不买！

岛上东南有个突出去的角，丰水期估计会被淹没，所以被围栏围了起来。但为了看远处的鸭子，我翻过围栏，走了出去。

上午去了伏见稻荷大社，兴味索然。上了山，到处都是小神社，倒是有点像麦荔枝跟碧山的融合：麦荔枝有徒步的森林，碧山有坟地，

## 结语 {#结语}

想不到这篇游记从2025年12月8日开始写，拖到2026年2月6日才写完。写得实在有点累，主要是整理照片有点麻烦，下次旅游估计会更麻烦，
因为可能会带相机。Anyway 下次再说。

这次跟东京（见[《东京也无非是这样》](https://dantezy.xyz/posts/tokyo-202507/)）因为已经是冬天了，京都天黑得很早，5点之后我走在街上，会觉得很落寞。
而且这种落寞是无处可排遣的，我当时希望在我身边的人不是现实存在的任何一个人，甚至即使我能召唤出岚山步鸟[^fn:12]，
我依然会觉得寂寞。这种漂泊的感觉无法排解，除非我在京都长大，京都成了我的故乡，但这样我就不是我的。
这就是永远的彼岸。
