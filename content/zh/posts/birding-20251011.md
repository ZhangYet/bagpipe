约了我的朋友亮哥带我去 [Tampines Eco Green](https://maps.app.goo.gl/okSW6f3hqKZnvnYDA) 观鸟。我9月傍晚也来过一次，
但技术不行，只看到寥寥几种鸟。所以特意约亮哥一起，跟伊学一下观鸟的技巧。

我们从南门出发，向东走，路上遇到一只噪鹃(Asian Koel, Eudynamys scolopaceus)。
我第一次见噪鹃是在 Clementi 楼下，一群大爷大妈围着一只黑色的鸟看，远看像乌鸦，
但是那对红眼很突出，回去查了一下才发现是噪鹃。9月份我单独来的那次也在树上看到一只。
噪鹃实在很吵，一叫就盖住其他声音，深圳的朋友深受其害。可惜的是我这三次看到的都是雄性成鸟。
红眼和黄色的喙是显著的特征。这只噪鹃在树丛深处，如果不是亮哥指出，我不会看留意到。

{{< figure src="/ox-hugo/asian_koel_female.png" caption="<span class=\"figure-number\">Figure 2: </span>雌性噪鹃/未成鸟雄鸟" >}}

去到 Turtle pond 我们还看到一条蛇从水里游过去，如果我没看错，这是一条有毒的蛇，
去徒步的朋友要多小心。这个小水塘里面确实有很多乌龟。在水塘边，
我们看到一只鹊鸲(Oriental Mappie-Robin, Copsychus saularis)和一只普通翠鸟
(Common Kingfisher, Alcedo atthis)。

{{< figure src="/ox-hugo/oriented_mappie_robin_male.jpeg" caption="<span class=\"figure-number\">Figure 3: </span>雄性鹊鸲" >}}

{{< figure src="/ox-hugo/oriented_mappie_robbin_female.jpeg" caption="<span class=\"figure-number\">Figure 4: </span>雌性鹊鸲" >}}

我们看到的鹊鸲是雄性的。

{{< figure src="/ox-hugo/common_kingfisher.jpeg" caption="<span class=\"figure-number\">Figure 5: </span>普通翠鸟" >}}

普通翠鸟雌雄区别不大，我们看到那只还挺年轻的，背上还不是一片碧蓝。

Turtle pond 旁边有一棵树，光秃秃的，不知道品种，上面聚集了一群绯胸鹦鹉
(Red-breasted Parakeet, Psittacula alexandri)、红领绿鹦鹉(Rose-ringed Parakeet, Psittacula krameri)
和亚洲辉椋(liáng)鸟(Asian Glossy Starling, Aplonis panayensis)。

查了一下其实绯胸鹦鹉是外来物种，见[狮城新闻](https://www.shicheng.news/zh-hant/v/jM6jV)。之前看到绯胸鹦鹉出现在蔡厝港的新闻想过去看看，
没有看到。后面我们还看见一只占据了一处树洞的绯胸鹦鹉。

{{< figure src="/ox-hugo/red_breasted_parakeet.jpeg" caption="<span class=\"figure-number\">Figure 6: </span>绯胸鹦鹉，雄性的喙是红色的，雌性是黑色的" >}}

红领绿鹦鹉看得比较多，在我家附近商场外的树也看到过（下午6点半左右会有一群飞过来）。

{{< figure src="/ox-hugo/rose_ringed_parakeet.jpeg" caption="<span class=\"figure-number\">Figure 7: </span>红领绿鹦鹉，但雌性是没有那个红领的" >}}

亚洲辉椋鸟(Asian Glossy Starling, Aplonis panayensis)长得像乌鸦，但是红眼。
也像噪鹃，但是喙是黑的，体型略小，有泛蓝的金属光泽（雄性噪鹃也有光泽，但是泛黑）。

{{< figure src="/ox-hugo/asian_glossy_starling.jpeg" caption="<span class=\"figure-number\">Figure 8: </span>亚洲辉椋鸟" >}}

随后我们向北面沿 Diversity Trail 走，路上亮哥看见了金背三趾啄木鸟，但是我没对焦上，
没看到。到 Freshwater pond 我们转向东边。路上看到了冠斑犀鸟(Oriental Pied Hornbill, Anthracoceros albirostris)。

{{< figure src="/ox-hugo/oriental_pied_hornbill.jpeg" caption="<span class=\"figure-number\">Figure 9: </span>冠斑犀鸟" >}}

冠斑犀鸟以喙上冠的大小区别雌雄，在此之前我见过两次冠斑犀鸟，2024年4月那次我拍了照片。
现在翻回来，应该是雌性的。

{{< figure src="/ox-hugo/oriented_pied_hornbill.jpg" caption="<span class=\"figure-number\">Figure 10: </span>雌性冠斑犀鸟" >}}

应该是在这一路上，我在一群白眉黄臀鹎中看到一只红耳鹎(Red-whiskered Bulbul, Pycnonotus jocosus)。
耳边的红点在十倍望远镜下只有一点点。鹎鸟头顶这撮冠羽是挺明显的特征，小小的很可爱。

{{< figure src="/ox-hugo/red_whiskered_bulbul.jpeg" caption="<span class=\"figure-number\">Figure 11: </span>红耳鹎" >}}

路上亮哥还给我指了北灰鹟(Asian Brown Flycatcher, Muscicapa dauurica)。
这种灰色的小鸟刚从北方飞来过冬，它们在茂密的枝条上活动，如果没有懂行的人指出，
我肯定不会留意到。而且现在让我描述它的特点我也说不出来。

{{< figure src="/ox-hugo/asian_brown_flycatcher.jpeg" caption="<span class=\"figure-number\">Figure 12: </span>北灰鹟" >}}

向东走到公园的边沿，我们沿着公园外围的 Forest Trail 往东北走。走到尽头再折向西边，
应该是在 Viewing Platform 里面看到黄胸织雀(Baya Weaver, Ploceus philippinus)——我们早看到它们精巧的巢了。
在这个地方我们还遇到一个手持长焦镜头的 Uncle ，Uncle 给我们推荐了 Singapore Quarry,
可以看到鹰，还给我们分享了伊拍到的凤头蜂鹰。

{{< figure src="/ox-hugo/baya_weaver_femele.jpeg" caption="<span class=\"figure-number\">Figure 13: </span>黄胸织雀" >}}

{{< figure src="/ox-hugo/baya_weaver_male.jpeg" caption="<span class=\"figure-number\">Figure 14: </span>繁殖期的黄毛雄性黄胸织雀和精巧的巢" >}}

不知道是不是因为不在繁殖期，看到的黄胸织雀都没有黄毛。

之后我们就沿着 Diversity Trail 回去了，也差不多十一点了。除了上面写到的鸟，
当天看到黑枕黄鹂（包括一只成年的和一只未成年的）、红原鸡和白胸苦恶鸟。这些鸟都太常见了，
就不另外记录了。
