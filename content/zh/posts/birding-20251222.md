本篇观鸟笔记涉及2025年12月的四次观鸟，这次之所以四次观鸟合并作一次写，
主要的原因有两个：

1.  观察到的新（或者说有趣的）鸟种没那么多了，毕竟我人还在新加坡。
2.  我懒。

另外本月观鸟的装备也变了，望远镜我都是用胜途 lamda 8x25 袖珍望远镜[^fn:1]。
手机我趁双十二买了 Pixel 10 pro ，它有5倍光学变焦，虽然不强吧，但我之前的 Pixel 7a
是没有光学变焦的[^fn:2]。另外这次观鸟里面我会用到 Pixel 10 pro 的算法补全。
所以本文会有很多很糊，但是能看出鸟类特征的照片（标题标注 Pixel ）。


## Tampines Eco Green 20251214 {#tampines-eco-green-20251214}

这次观鸟主要目的是要把我的旗云带给亮哥，让伊帮我带回国内保修（左侧目镜镜筒转不出来了）。

去到 Turtle pond, 里面有五只水獭在开淡水鱼自助餐，其中有[一只吃鱼吃得好开心](https://t.me/danteslimbo/5839)，把我看饿了。

{{< figure src="/ox-hugo/otta-eating-fish.jpg" >}}

但也许正是因为有这水獭一家子，Turtle pond 附近都没有鸟了。Turtle pond 旁边的高树上，
可能是木棉树，上面依然是一群绯胸鹦鹉，旁边树里有一只黑枕黄鹂扑棱了一下，我拍了一下，
小试了一把 Pixel 大于10倍的镜头补全。

{{< figure src="/ox-hugo/black-naped-oriole.png" caption="<span class=\"figure-number\">Figure 1: </span>这是不是就是过曝啊(Pixel)？" >}}

往里面走，在一棵秃树上看到了一只白领翡翠(Collared Kingfisher, Todiramphus chloris)，
我第一次在新加坡看到白领翡翠是今年2月在巴西立公园，那个时候还没有正式记录观鸟。
而且当时我会用 Pixel 7a 加望远镜去拍照，很难拍清。

{{< figure src="/ox-hugo/collared-kingfisher1.png" caption="<span class=\"figure-number\">Figure 2: </span>虽然补全得像油画(Pixel)" >}}

{{< figure src="/ox-hugo/collared-kingfisher2.png" caption="<span class=\"figure-number\">Figure 3: </span>但特征还是出来了(Pixel)" >}}

走走停停，走到 Freshwater Pond 的时候，亮哥来了，我回到 Turtle pond 跟伊回合，一块看水獭。

{{< figure src="/ox-hugo/otto-family.jpeg" caption="<span class=\"figure-number\">Figure 4: </span>水獭一家五口(Pixel)" >}}

有一对冠斑犀鸟飞到那棵聚满绯胸鹦鹉的树上，估计这群绯胸鹦鹉跟这对犀鸟不熟，
大部分鹦鹉都飞走了，我觉得挺奇怪的，虽然犀鸟很大，但毕竟不是猛禽，何必呢?

{{< figure src="/ox-hugo/hornbill1.jpg" caption="<span class=\"figure-number\">Figure 5: </span>冠斑犀鸟(Pixel)" >}}

{{< figure src="/ox-hugo/hornbill2.jpg" caption="<span class=\"figure-number\">Figure 6: </span>这张算法补得有点夸张了，脖子上都补出绒毛了(Pixel)" >}}

随后我们北上，在 Freshwater Pond 转向东，在那条小路上我看到远处有一只正对着我的翠鸟，很遗憾没拍到照片，
亮哥认为这是一只普通翠鸟，我认为是一只赤翡翠，因为我看到它的头和喙都很红，但诡异的是胸前有一块白，
我现在只能存疑。

<span class="timestamp-wrapper"><span class="timestamp">&lt;2025-12-26 Fri 16:26&gt; </span></span> 重新翻图鉴，我感觉更可能是白胸翡翠，最突出的是红色的喙。

{{< figure src="/ox-hugo/map.png" caption="<span class=\"figure-number\">Figure 7: </span>谁这么无聊给起了个蔡徐坤森林啊" >}}

我们向东走到底，折向西南，路上看到一只黑鸣鹃鵙(Pied Triller, Lalage nigra)，也是很可惜，没有照片。
这应该是成鸟，黑白很分明。还有一只伯劳(Brown Shrike, Lanius cristatus)。伯劳看着挺可爱的，居然是猛禽，
《休伯利安》里面满身钢铁荆棘的机器人也叫伯劳。《诗经·七月》里面说「七月鸣鵙」，鵙就是伯劳，
果麦文化出了一本骆玉明注解、细井徇绘图的《诗经》，里面画的伯劳虽然是水墨画，但是特征很明显。
话说回来，伯劳在中国古典文学里面最出名的应该是萧衍的「东飞伯劳西飞燕」。

{{< figure src="/ox-hugo/brown-shrike.jpg" caption="<span class=\"figure-number\">Figure 8: </span>伯劳那个钩嘴被灌木挡住了，恨！(Pixel)" >}}

大概就是在地图标注蔡徐坤森林那地方，有一棵树，聚集了一群蓝喉蜂虎(Blue-throated Bee-eater, Merops viridis)，
树上我们还看到一对红嘴黑身的鸟，亮哥认为这是三宝鸟(Oriental Dollarbird, Eurystomus orientalis)。
我后来翻 [singaporebirds](https://singaporebirds.com/species/) 的图鉴，感觉也可能是翠金鹃(Asian Emerald Cuckoo,
Chrysococcyx maculatus)。

{{< figure src="/ox-hugo/blue-throated-bee-eater.jpeg" caption="<span class=\"figure-number\">Figure 9: </span>这蓝喉蜂虎糊得有点过分了(Pixel)" >}}

{{< figure src="/ox-hugo/dollarbird.jpg" caption="<span class=\"figure-number\">Figure 10: </span>我觉得真的更像翠金鹃，因为三宝鸟的嘴更像钩子，但拍得这么糊，没办法了(Pixel)" >}}

最后我们还看到一只普通翠鸟，这次 Pixel 拍的照片就不知所谓了。

{{< figure src="/ox-hugo/common-kingfisher.jpg" caption="<span class=\"figure-number\">Figure 11: </span>所以说要有好照片，一靠光线二别抖(Pixel)" >}}


## Jurong Lake Gardens 20251220 {#jurong-lake-gardens-20251220}

Jurong Lake Gardens 离我家只有一站路，18、19号的时候我们其实计划去新山，但想不到碰上假期高峰，关口大堵车，
所以出游计划 GG，我早上起来就去 Jurong Lake Gardens 观鸟。

从 Chinese Garden 站出来绕过工地，在进入 Chinese Garden 的桥上，我看到一只黄苇鳽(Yellow Bittern, Botaurus sinensis)
在水中做一字马。

{{< figure src="/ox-hugo/yellow-bittern.jpg" caption="<span class=\"figure-number\">Figure 12: </span>黄苇鳽(Pixel)" >}}

过了桥看到有两只猛禽，但是很快飞出视野了。
湖边有一棵树上面聚集了一群栗喉蜂虎(Blue-tailed Bee-eater, Merops philippinus)，
因为光线原因，它们的 blud tail 看起来一点也不蓝（转过去才勉强有点蓝色），
但是尾巴上細針狀中央羽毛很明显，我凭这个识别鸟种[^fn:3]。虽然丫叫 bee-eater, 但我看见其中一只大快朵颐地吃蜻蜓。

{{< figure src="/ox-hugo/blue-tailed-bee-eater.jpeg" caption="<span class=\"figure-number\">Figure 13: </span>尾巴看起来不蓝，喉部的颜色也看不清(Pixel)" >}}

{{< figure src="/ox-hugo/blue-tailed-bee-eater1.jpeg" caption="<span class=\"figure-number\">Figure 14: </span>转过一个角度就能看到一点蓝色了(Pixel)" >}}

转过去有两组鸟太快，来不及识别。但是有一只鸟站在高高的树枝上，通体发绿，有一个粗短的喙，回来翻图鉴，
我判断为斑头绿拟啄木鸟(Lineated Barbet, Psilopogon lineatus)。

{{< figure src="/ox-hugo/lineated-barbet.jpeg" caption="<span class=\"figure-number\">Figure 15: </span>丫站那么高，我也没办法(Pixel)" >}}

一路走过去，在花丛中看到一只朱背啄花鸟(Scarlet-backed Flowerpecker, Dicaeum cruentatum)，
这是当天最大的惊喜，我喜欢这种颜色鲜艳的鸟，虽然当场识别不出来，但是把图发给亮哥，
人家一眼看出来了。

{{< figure src="/ox-hugo/scarlet-backed-flowerpecker.jpeg" caption="<span class=\"figure-number\">Figure 16: </span>朱背啄花鸟，真的很漂亮(Pixel)" >}}

随后看到草地上有两只鸟在厮打（我还以为其中一只鸟有毛病），我凑近拍照片的时候它两还在玩，知道旁边窜出一条蜥蜴。
Merlin 识别为灰缝叶莺，我总感觉体型比灰缝叶莺大。

{{< figure src="/ox-hugo/two-birds.jpeg" caption="<span class=\"figure-number\">Figure 17: </span>有认识的朋友给说说吗？(Pixel)" >}}

后面我回到入口那座桥（看到黄苇鳽）那座，有个 Uncle 和一个姑娘用力往旁边那棵繁茂的大树看。Uncle
热情地给我指出了树里面的斑扇尾鹟（只看到尾巴）和红翅凤头鹃(Chestnut-winged Cuckoo, Clamator coromandus)，后者也是明星鸟啊。
上小红书搜就能搜到。

{{< figure src="/ox-hugo/chestnut-winged-cuckoo.png" caption="<span class=\"figure-number\">Figure 18: </span>红翅凤头鹃，图片来源： <https://singaporebirds.com/species/chestnut-winged-cuckoo/>" >}}

Jurong Lake Gardens 是几个人工花园，本来我对这趟观鸟之旅不报希望，但看到几种新鸟，值了。另外还看到一只鹰，可惜识别不出来。

{{< figure src="/ox-hugo/hawk.jpeg" caption="<span class=\"figure-number\">Figure 19: </span>翅膀末端是白色的，会是凤头蜂鹰吗？(Pixel)" >}}


## Marina Grove, Jurong Lake Gardens 20251221 {#marina-grove-jurong-lake-gardens-20251221}

Marina Grove 据说是圣淘沙下面一丛小灌木，Google Map 给我指了一条路出来，但是小红书说其实是 Supertree Grove 给指了条路。
我也不知道对不对，我大概九点多到 Gardens By the Bay MRT 出站，车站周围的鸟鸣吵得紧（几乎都是麻雀）。

可能我找的地方不对，最后只看到一只鹩哥(Common Hill Myna, Gracula religiosa)，一只白领翡翠，还有一只鹊鸲（我还抓到这只鹊鸲随地大小便）。

{{< figure src="/ox-hugo/common-myna.jpeg" caption="<span class=\"figure-number\">Figure 20: </span>Common Myna 现在也不怎么 Common 了，我来新加坡那么久也就见着两回(Pixel)" >}}

{{< figure src="/ox-hugo/collared-kingfisher3.jpeg" caption="<span class=\"figure-number\">Figure 21: </span>白领翡翠(Pixel)" >}}

{{< figure src="/ox-hugo/mapie-robin.jpeg" caption="<span class=\"figure-number\">Figure 22: </span>鹊鸲(Pixel)" >}}

上午铩羽而归，所以下午决定再去一次 Jurong Lake Gardens，毕竟六月我就在那里见到白额燕鸥(Little Tern, Sternula albifrons <https://t.me/danteslimbo/4938>) 。
但这次我失望了，20号在灌木丛中翻飞的各种鸣禽都隐匿了，只有鸽子八哥和乌鸦在开会。河边倒是见到一只白领翡翠。

回去的时候，我绕着湖去 Lakeside 站，岸边看到另一只黄苇鳽, Merlin 把它识别成了栗苇鳽。

{{< figure src="/ox-hugo/yellow-bittern1.jpeg" caption="<span class=\"figure-number\">Figure 23: </span>黄苇鳽，脖子上还真的有点红(Pixel)" >}}

郁闷的是，没能再看见白额燕鸥。。


## Jurong Lake Gardens 20251225 {#jurong-lake-gardens-20251225}

12月25日是圣诞假期，但这周我值班，为了防止国内同事忽然找我，我也不敢去更远的地方，于是只能去 Jurong Lake
Gardens 了。这地方之于我也差不多像 Tampines Eco Green 之于亮哥了。

这天我去得早，大约8点多就到了，至少三个人长枪短炮对着发现红翅凤头鹃那棵树拍，我看了一下，
只看见一只黑枕黄鹂的未成鸟。唉，我之前就说过我有点势利眼，要是凤头鹃还值得一看，黄鹂就敬谢不敏了，
然后这天我真看到很多黄鹂。

{{< figure src="/ox-hugo/black-naped-oriole1.jpeg" caption="<span class=\"figure-number\">Figure 24: </span>隔叶黄鹂空好声(Pixel)" >}}

这次我决定换一条路，从裕华园那个塔[^fn:4]下面的广场穿过去，那个广场旁边树上有一群栗喉蜂虎。

穿过去，在荷花池旁边的灌木丛看到了两只栗腹文鸟(Chestnut Munia, Lonchura atricapilla)。图如下，肯定看不清，因为被灌木遮掩了。
但是我用望远镜看的时候可以清楚看到它的锥型的短喙、黑色的头和栗色到发红的背，特征很明显。
我就喜欢这种特征明显的鸟。

{{< figure src="/ox-hugo/chestnut-munia.jpeg" caption="<span class=\"figure-number\">Figure 25: </span>将就点看吧，没有这么茂密的树丛，文鸟不来，但树丛茂密了容易挡住鸟(Pixel)" >}}

接下来我看到了鹊鸲和灰缝叶莺(Ashy Tailorbird, Orthotomus ruficeps)。鹊鸲不是罕见的鸟，但能拍得那么高清就很难得。
用5倍光学焦距镜头的手机拍灰缝叶莺拍得那么清晰就更难得了。就冲今天看到的栗腹文鸟，
拍到那么清晰的鹊鸲和灰缝叶莺我觉得值了，余之生涯一片无悔。

{{< figure src="/ox-hugo/mapie-robin1.jpeg" caption="<span class=\"figure-number\">Figure 26: </span>高清鹊鸲(Pixel)" >}}

{{< figure src="/ox-hugo/ashy-tailorbird.jpeg" caption="<span class=\"figure-number\">Figure 27: </span>灰缝叶莺(Pixel)" >}}

最近 ebird 上的观鸟列表有看到一些之前我没有看到过的 waterhen 的，所以我沿着湖边（这个湖也没个名字）
向 Lakeside MRT 走去，希望能看到一些涉禽——结果除了一只苍鹭（这苍鹭一点都不怕人，它飞到打捞湖面垃圾的小船上，跟小船一块走），啥都没看见。快到 Lakeside 的时候，
我改变主意，又回去 Chinese Garden 站。路上看到一只斑扇尾鹟。

{{< figure src="/ox-hugo/fantail.jpeg" caption="<span class=\"figure-number\">Figure 28: </span>虽然最重要的扇尾没拍到，但是脖子这白环跟白色胸部也很突出了(Pixel)" >}}

回到曾经看到凤头鹃那棵树附近（因为我也想再看一眼凤头鹃），有个印度哥们举着大镜头，也在找它，
伊应该听到了声音，不过没找到，但我已经累了，所以就道别回家了。
